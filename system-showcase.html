مكتبة تدريبية شاملة في انتظارك

بعد الاشتراك، ستحصل على وصول كامل لمكتبة فيديوهات تدريبية تغطي كل ميزة في النظام بالتفصيل. بالإضافة إلى جلسات تدريب مباشرة مع فريقنا لضمان استفادتك القصوى من النظام.

01

لوحة التحكم الذكية

لوحة تحكم ذكية تعرض أهم الإحصائيات في لمحة واحدة، مع رسوم بيانية تفاعلية لمتابعة أداء العيادة بشكل لحظي.

  • إحصائيات اليوم والشهر والسنة
  • متابعة الإيرادات والمصروفات لحظياً
  • عرض المواعيد والمهام المعلقة
  • مؤشرات أداء الأطباء والخدمات
  • تنبيهات ذكية لمخزون الصيدلية
  • رسوم بيانية تفاعلية احترافية
02

نظام المواعيد والكشوفات

نظام مواعيد مرن يتيح إضافة، تعديل، تأكيد أو إلغاء المواعيد بسهولة، مع إشعارات تلقائية للمرضى والأطباء.

  • إضافة وتعديل المواعيد بمرونة كاملة
  • إشعارات تلقائية عبر البريد والواتساب
  • جدولة الورديات للأطباء والعاملين
  • متابعة الحالات المزمنة وجدولة المراجعات
  • تقويم شهري/أسبوعي/يومي تفاعلي
  • إدارة قوائم الانتظار بذكاء
03

السجلات الطبية والروشتات

إدارة شاملة للمرضى والملفات الطبية والسجلات السابقة، مع كتابة الروشتات الإلكترونية وحفظها تلقائياً.

  • ملف طبي شامل لكل مريض
  • التاريخ المرضي والسجلات السابقة
  • كتابة الروشتات إلكترونياً
  • ربط مباشر بين الطبيب والصيدلية
  • طلب التحاليل وتسجيل النتائج
  • إرفاق الأشعة والتقارير الطبية
04

نظام الفواتير والمحاسبة

نظام فواتير ومدفوعات احترافي مع دعم الضرائب والخصومات، وتتبع دقيق للإيرادات والمصروفات.

  • إصدار فواتير احترافية مع الضرائب
  • دعم جميع طرق الدفع (كاش - فوري - فيزا - محافظ)
  • تتبع الإيرادات والمصروفات التفصيلية
  • نظام الرواتب والمصروفات التشغيلية
  • طباعة أو إرسال الفواتير إلكترونياً
  • تقارير مالية شاملة بصيغ PDF/Excel
05

التقارير والإحصائيات المتقدمة

لوحة تقارير تفاعلية لعرض الأداء والإحصاءات، مع إمكانية إنشاء وتحميل التقارير بصيغ متعددة.

  • تقارير مالية تفصيلية (إيرادات - مصروفات)
  • تقارير أداء الأطباء والخدمات
  • تقارير التحاليل والأدوية والمخزون
  • تصدير بصيغة PDF وExcel بضغطة واحدة
  • رسوم بيانية تفاعلية احترافية
  • تقارير مخصصة حسب الفترة والفئة
06

إدارة الأطباء والمستخدمين

إدارة كاملة للأطباء والمساعدين وجدولة الورديات، مع نظام صلاحيات دقيق لكل دور (طبيب - ممرض - كاشير - مدير).

  • إضافة وإدارة الأطباء والعاملين
  • جدولة الورديات والأوقات بسهولة
  • صلاحيات دقيقة لكل مستخدم
  • تتبع أداء الأطباء والمبيعات
  • إدارة التخصصات وأسعار الخدمات
  • حماية متقدمة للبيانات بتقنية SSL
07

نظام الصيدلية والمخزون

إدارة كاملة للمخزون والأدوية مع تنبيه عند انخفاض الكميات أو قرب انتهاء الصلاحية، مع دعم Batch Tracking.

  • إدارة كاملة للأدوية والمخزون
  • تتبع الدفعات (Batch Tracking) لكل صنف
  • تنبيهات تلقائية للكميات المنخفضة
  • تنبيه عند قرب انتهاء الصلاحية
  • إدارة الموردين والفواتير الشرائية
  • ربط مباشر مع الروشتات الطبية
